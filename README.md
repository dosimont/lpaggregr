# lpaggregr

## Dependencies

    Rcpp
    RcppArmadillo
    sqldf
    dplyr
    ggplot2
    reshape2
    RColorBrewer
    digest
    treemap
    gplots

## Install

    library(devtools);
    install_github("dosimont/lpaggregr");

## Load

    library(lpaggregr);

## Examples

    git clone https://github.com/dosimont/lpaggregr-example

Look at the example files `example_nemo_cluster.R` and `example_nemo_counter.R`
